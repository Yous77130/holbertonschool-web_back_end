# Python - Async

Projet d'introduction à la programmation asynchrone en Python avec `asyncio`.

## Description

Ce projet couvre les bases de l'asynchrone : la syntaxe `async`/`await`,
l'exécution de coroutines concurrentes, la création de tâches `asyncio`
et l'utilisation du module `random`.

## Fichiers

* `0-basic_async_syntax.py` : coroutine `wait_random` qui attend un délai
  aléatoire entre 0 et `max_delay` secondes et le retourne.

## Environnement

* Ubuntu 20.04 LTS
* Python 3.8
* pycodestyle 2.5.x
