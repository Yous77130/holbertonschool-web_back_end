# Python - Async Comprehension

Projet sur les generateurs asynchrones et les comprehensions asynchrones
en Python avec `asyncio`.

## Objectifs

* Ecrire un generateur asynchrone
* Utiliser les comprehensions asynchrones
* Annoter les types des generateurs

## Fichiers

* `0-async_generator.py` : generateur asynchrone yieldant 10 nombres
  aleatoires entre 0 et 10, avec une attente d'1 seconde a chaque iteration.

## Environnement

* Ubuntu 20.04 LTS
* Python 3.8
* pycodestyle 2.5.x
